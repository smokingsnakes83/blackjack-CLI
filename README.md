# blackjack-CLI v1.01 
	
<b>This game is based in the black jack game.</b>

<b>Install:</b><br/>

Open the terminal and type:<br>
'''
git clone https://github.com/smokingsnakes83/tblack-jack.git blackjack-CLI
'''
cd blackjack-CLI
make
sudo make install

<br><br>
<b>To uninstall:</b><br/>

In the terminal type:<br>
sudo make uninstall.
