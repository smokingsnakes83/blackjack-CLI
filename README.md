# Blackjack-CLI v1.01 
	
# This game is based in the black jack game.

## Install:

## Open the terminal and type:
```bash
git clone https://github.com/smokingsnakes83/tblack-jack.git blackjack-CLI
cd blackjack-CLI
make
sudo make install
```

# To uninstall:

## In the terminal type:
```
sudo make uninstall.
```
